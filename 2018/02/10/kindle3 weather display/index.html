kindle3 weather display

coding farm
2018-02-10 (Updated: 2018-02-11)
kindle, 趣物

作为一个技术小白，却有着一颗大牛的追求。前段时间看到题为《Kindle Weather Display》的一篇介绍如何让kindle不再吃灰的文章，我看看自己的kpw3，不舍得折腾，遂逛咸鱼，买了一个北京小姐姐的11年左右买的老古董。但是！真的很新。。。可以说是99新了，相当值得！感谢小姐姐！

充电将近40分钟成功开机。看到里面mysql、linux等等pdf，不禁笑了，kindle3原来的主人也是码农一位。

下面记录一下自己折腾一天的历程：
首先查阅github等网站，得到一个大概的思路：首先越狱，然后生成一张带有天气信息的图片，最后kindle显示出这张图片。生成图片的工作可以用server或者kindle本身，因为考虑到耗电以及自己刚好有条件，于是就按照cathay4t / kindle-weather的教程进行。这个大牛应该是国人，采用起来方便，基本可以不用修改。

操作步骤

server 准备

1、使用python2
由于centos6之前装了python3，虽然和python2共存了，但是输入 python命令出来的还是python3。

这个问题的原因呢在于之前的软链接做的有问题，阅读学习《CentOS 7 安装 Python3、pip3
》以及《linux中python2和python3环境搭建及共存》这两篇文章。

发现一些可执行文件/usr/bin/下ll查看python2文件，发现软链接没有对应文件python2.6而是python，更改后，正常。但是pip在python2中没能成功使用，虽然本次没有影响，但是问题仍需要解决。

kindle3 越狱

越狱也花了不少时间，查阅《折腾Kindle3-2011/09/18》和《kindle3的新手简单说说——Kindle3的经典“折腾” (2011-09-05 00:17:02)》以及《胡骊的blog-2016-03-11》

最新的破解包尝试四五次无果后，根据《Kindle 3 jailbreak + usbNetwork 破解流程》下载 kindle-3.2.1-jailbreak-0.5.1.zip (56.3 KB, 5921 views) 成功越狱。接着安装了最新的usbnetwork，在https://www.mobileread.com/forums/showthread.php?t=88004 上下载 kindle-usbnetwork-0.57.N-k3.zip (13.34 MB, 4441 views) 成功安装。
